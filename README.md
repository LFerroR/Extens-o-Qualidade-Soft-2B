# Extensão-Qualidade-Software-2B

Trabalho de Extensão Qualidade de Software 

Trabalho desenvolvido para atender aos requisitos da disciplina de Extensão em Qualidade de Software. O objetivo foi elaborar e documentar cenários e casos de teste aplicados a um sistema de controle de estoque, clientes, fornecedores e operações comerciais.

Foram definidos cenários abrangendo as principais funcionalidades do sistema, incluindo cadastros (Clientes, Fornecedores, Funcionários, Transportadoras e Produtos) e processos operacionais (Compras e Vendas).

Cada cenário contém três casos de teste — permitindo avaliar tanto o funcionamento esperado quanto o comportamento do sistema diante de entradas inválidas ou tentativas de ações duplicadas. A abordagem tem como propósito garantir a confiabilidade, consistência e qualidade das operações essenciais do sistema.
